Koui Genji Monogatari Text Database

A database publishing text data from the 'Koui Genji Monogatari' (Collated Genji Monogatari)

IIIF & TEI Viewer

View texts alongside IIIF images from the National Diet Library Digital Collection. Display high-resolution classical manuscript images and text data simultaneously.

TEI/XML

Browse TEI/XML files. Provides structured data with text encoding that follows international standards.

Linked Open Data

Browse text data with assigned URIs. Provides data in a format that follows Semantic Web standards and can be linked with other datasets.

DTS Viewer

Browse text data through the DTS (Distributed Text Services) API. Enables structural navigation of texts through standardized APIs.

About This Site

Text data was created and published based on the'Koui Genji Monogatari' (Collated Genji Monogatari)(Internet publication, copyright expired) available in the National Diet Library Digital Collection.

Each line is assigned a URI (Uniform Resource Identifier) and published as LOD (Linked Open Data). TEI/XML files linked with IIIF images are also provided.

If you find any errors in the data, please contact the University of Tokyo Academic Asset Archiving Promotion Office (General Affairs Division, University Library) at digital-archive [at] lib.u-tokyo.ac.jp.
